# 开发记录：前后端重构与蓝图风格升级（2026-07-19）

## 1. 重构背景与目标

随着“工程设计自动化工具台”在二院通信科室的逐步推广，原有的代码架构暴露出一些局限性：
1. **高耦合度**：原先的 `app.py` 中包含了大量的业务配置（如 19 种房屋参数的默认值、电缆沟规格、列表排序映射等）与底层解析代码，不仅显得臃肿，且业务参数的任何变动都需要修改后端逻辑。
2. **交互体验简陋**：原 UI 页面缺乏足够的视觉引导。特别是电缆沟尺寸的改动不够直观，Excel 数据表的导入要求缺乏清晰指示，导致用户容易填错格式。
3. **报错不友好**：当模板中发生 Jinja2 语法错误或文件读写冲突时，系统直接向前端暴露复杂的 Python 异常 Traceback 堆栈，用户难以理解，无法自主定位问题。

本轮升级通过**后端分层重构与业务外部化**和**前端蓝图暗黑主题化与交互升级**，彻底解决了上述痛点。

---

## 2. 后端重构：配置外部化与分层架构

后端结构从“一竿子插到底”的单文件，演进为了配置、服务、控制器清晰解耦的规范化分层架构：

### 2.1 业务配置外部化 (`config/business_settings.json`)
将硬编码在代码中的静态业务规则，全部抽离至 `config/business_settings.json`：
- **机房默认指标**：包含通信站、信号楼等 19 种机房的面积、用电量、位置等参数。
- **电缆沟尺寸**：电缆沟/电缆槽宽深的基准参数。
- **数据表结构配置**：Excel 里程表的数据映射字段名、表头定义。

### 2.2 核心模块重解耦
- **配置加载层 (`services/config_loader.py`)**：负责读取外部 JSON 配置文件，并内置了一套降级默认配置（Fallback）。即使 JSON 文件丢失或损坏，系统仍旧可以降级运行而不会发生崩溃。
- **文档生成服务 (`services/docx_renderer.py`)**：剥离出了 QoS 审查卡片和互提资料文档的具体渲染流程。包含：前端请求参数与后端配置默认值的深度合并、富文本换行符（`\a` 软回车）的安全过滤转换、机房在不同专业表中的动态排版序号计算等。
- **XML 适配层 (`services/xml_helper.py`)**：通过 `python-docx` 接口实现对 Word 二进制包中 XML 文档的底层过滤和修改。一是实现了自动清洗拼写检查断裂标签的功能，二是处理 Excel 里程表数据到 Word 内部指定表格位置的节点动态追加。

### 2.3 异常业务化机制 (`translate_error`)
在 `app.py` 路由出口中新增了异常过滤器：
- **Jinja2 模板语法报错**：拦截 `TemplateSyntaxError`，转换为“模板语法错误：检测到 Word 模板中的 Jinja2 标签语法不正确，可能存在未配对的 %if% 或 %for% 段落标记”，并附带错误位置。
- **模板文件冲突**：拦截 `PermissionError`，友好提示“文件访问受限：请确保模板文件未被其他程序（如 Word）独占打开，且具备读写权限”。
- **文件缺失**：拦截 `FileNotFoundError`，友好提示“模板文件缺失：请确认 templates_docx/ 目录下的模板完整性”。

## 3. 前端升级：蓝图暗黑风格与人机交互优化

为了突出系统的“工程自动化”专业定位，前端的视觉和微交互进行了深度优化：

### 3.1 蓝图暗黑主题 (Blueprint Dark Mode)
- 重新设计了 `static/css/style.css` 风格体系。页面采用科技蓝底色配合浅色线条网格，模拟“晒蓝图”的视觉体验，极大提升了产品的工业级品质感。
- 引入顺滑的过渡动画与微交互 hover 效果，使操作反馈更灵敏。

### 3.2 动态 SVG 电缆沟剖面预览
- 在互提资料生成器中，基于 SVG 动态计算并渲染了一幅**电缆沟断面示意图**。
- 当工程师修改电缆沟的主沟宽、主沟深、线槽尺寸等参数时，SVG 的几何路径、尺寸标注线与文本会跟随输入框即时重绘，以“所见即所得”的方式帮助工程师直观把控电缆沟比例，避免数值填写离奇偏差。

### 3.3 QoS 片段快捷输入抽屉 (Snippets Drawer)
- 在 QoS 页面新增了一个侧边可收缩的“常用意见抽屉”。
- 抽屉内嵌了常用审查段落、复核意见及常用词条（如“确认已修改”、“各级意见已落实”等）。工程师只需鼠标点击，即可一键填入或追加到对应的文本框中，降低了打字录入的负担。

### 3.4 Excel 数据导入悬浮 HUD
- 在过轨表、引下位置表等 Excel 导入框旁，增加了悬停 HUD 指示器。
- 鼠标滑过时会唤醒悬浮气泡，清晰展示导入所需的 Excel 表头列名、数据格式以及空值填报规范，防止导入错误导致的生成失败。
