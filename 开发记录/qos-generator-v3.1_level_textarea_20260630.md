# QoS 生成器 v3.1 — 终审级别三选一 + 意见文本框

## 需求
1. 终审级别增加"所级"选项（所级/院级/公司级三选一），生成的表格截止到对应级别
2. 复核意见、审查意见从"一行一条"的列表式改为文本框输入

## 改动

### 后端 (app.py)
- **新增 `remove_tables_by_level` 函数**：根据终审级别删除多余的 `<w:tbl>` 表格
  - 所级：删除表格4(院级)和表格5(公司级)，保留3张表
  - 院级：删除表格5(公司级)，保留4张表
  - 公司级：全部保留，5张表
- **意见数据格式变更**：从 `review_opinions: [列表]` 改为 `review_opinions_text: "多行文本"`，后端直接将文本替换到占位符，不再做序号格式化
- 删除了 `format_opinions` / `format_opinions_cn` 函数（不再需要）
- 删除了 `replace_placeholder_across_runs` 函数（冗余）

### 前端 (index.html)
- **终审级别**：增加"所级"单选项，默认"院级"
- **级别提示**：切换级别时显示"生成表格：复核卡片 + 所室级审查 [+ 院级审查] [+ 公司级审查]"
- **动态显示/隐藏**：
  - 所级：只显示复核 + 所室级
  - 院级：显示复核 + 所室级 + 院级
  - 公司级：全部显示
- **意见输入**：全部改为 `<textarea>`，用户自己换行编写意见内容
- 删除了 `addOpinion` / `removeOpinion` / `renumber` 等列表管理 JS 代码
